# Student Data Search using Binary Search

## Project Description
This project is a Python program that searches for student details using the **Binary Search Algorithm**. The user enters the student's name, and the program quickly finds and displays the student's information if it exists in the dataset.

##  Features
- Search student details by name
- Uses Binary Search for fast searching
- Displays:
  - Student Name
  - Roll Number
  - Father's Name
  - Grade
- User-friendly console interface

# Technologies Used
- Python 3
- Binary Search Algorithm
